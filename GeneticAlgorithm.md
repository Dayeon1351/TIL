 * 유전 알고리즘
 > 적자생존 이론을 기본 개념으로 한다.  
 > 문제에 대한 가능한 해들을 나열한 뒤, 점점 유전자들을 변화시켜 정확도가 높고 좋은 해들을 만들어 낸다. 
  + 유전자 : 문제의 해들 
  + 진화 : 유전자들을 변형시켜 좋은 해를 얻는 것
 > => 더 좋은 답을 찾아 가기 위해 진화를 모방한 탐색 알고리즘이라고 할 수 있다.
  
  + 과정
  1. 초기화 (Initialize)
  2. 선택 (Selection)
  3. 교차 (Crossover)
  4. 변이 (Mutation)
  5. 대치 (Replace)
  6. 반복 (Loop)
  7. 종료
